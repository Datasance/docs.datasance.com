# Deploy Keycloak

## Download Realm Configuration

You can download preconfigured realm import json for Datasance PoT with below command. Replace `newRealmValue` with the you would like to give for your realm.

```bash
wget -q -O - "https://example.com/data.json" | sed 's/\$realm/newRealmValue/g' > datasance-pot-ralm.json
```
##### Example

```bash title="Download preconfigured Datasance PoT realm configuration and assing realm-name: foo "
wget -q -O - "https://example.com/data.json" | sed 's/\$realm/foo/g' > datasance-pot-ralm.json
```
